# Birdor — 面向开发者的工具与笔记

**Birdor** 是一个安静、友好、专注的开发者工具站。
我们构建 **实用的工具**、**清晰的工程笔记**、和 **易读的技术文档**，帮助开发者理解现代软件，而无需面对噪音、广告或混乱交互。

Birdor 的理念：

* **冷静（Calm）**：无广告、无弹窗、无干扰
* **快速（Fast）**：轻量、前端优先、无需后端依赖
* **实用（Practical）**：源自真实工程场景
* **诚实（Honest）**：明确指出利弊、给出稳定做法
* **开发者优先（Developer-first）**：为每天写代码的人设计

访问站点 → **[https://birdor.com](https://birdor.com)**

## Birdor 包含哪些内容？

### 1. Birdor Tools（工具集）

轻量、纯前端、隐私友好的开发者工具集合。

已上线的工具包括：

* **JSON 格式化与结构查看器**
* **JWT 解码与签名验证工具**
* **URL 解析与 Query 编辑器**
* **正则表达式测试器**
* **时间戳转换器**
* **进制转换（2/10/16）**
* **Hash 计算（MD5/SHA）**
* **Base64 编解码**
* ……持续扩展中

Birdor 的工具遵循三个原则：

* **完全客户端运行**
* **状态保留在本地**
* **轻量、冷静、可预测的体验**

工具入口：
[https://birdor.com/tools](https://birdor.com/tools)

## 项目目标

Birdor 希望成为：

* 一个 **冷静的开发者工具箱**
* 一个 **现代软件工程手册**
* 一个 **面向个人开发者的微型 SaaS 框架（未来）**
* 让你快速理解某件事“如何工作”与“为什么这样工作”的地方

没有市场营销语言，没有虚夸的叙事，没有干扰。
只有清晰工程。
